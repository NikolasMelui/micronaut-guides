7.1 Google App Engine Gradle Plugin

To deploy to App Engine, we are going to use the Google App Engine Gradle Plugin.

Add the plugin as a buildscript dependency:

build.gradle
buildscript {
    repositories {
        mavenCentral()
    }

    dependencies {
        classpath 'com.google.cloud.tools:appengine-gradle-plugin:2.0.0-rc4'
    }
}

Apply the plugin and specify a deployment version and the project ID. When you created the Project in the Google Cloud Dashboard you got the Project ID. Notice I didnâ€™t use dots on the version number.

build.gradle
apply plugin: 'com.google.cloud.tools.appengine'

version "0-2"
group "example.micronaut"

ext {
    appEngineProjectVersion = project.version
    appEngineProjectId = 'calm-history-228704'
}

Then configure the deployment properties:

build.gradle
appengine {
  stage.artifact = "${buildDir}/libs/${project.name}-${project.version}-all.jar"
  deploy {
      projectId = appEngineProjectId
      version = appEngineProjectVersion
  }
}
